# delight-charts

delight图表库

开发环境准备

推荐node14.19.3, 其他版本对vitepress 3.0的兼容性略差
推荐7.3.0, 其他版本对node 14的兼容性略差
nvm use v14.19.3
npm i -g pnpm@7.3.0

## 贡献

```shell
pnpm i
```

运行图表库组件demo和开发图表组件

```shell
pnpm dev
```
构建图表库

```shell
pnpm build
```

运行文档和开发文档组件

```shell
pnpm docs:dev
```

构建文档

```shell
pnpm build:doc
```


build 使用 v18.18.2
docs:dev 使用 18.20.4
